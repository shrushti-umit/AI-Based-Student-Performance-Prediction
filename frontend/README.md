# Frontend

This folder contains the frontend (user interface) of the AI-Based Student Performance Prediction project.

## Files

- `index.html` – Main webpage for user interaction.
- `style.css` – Stylesheet for designing the user interface.
- `script.js` – JavaScript file for handling user input and prediction logic.

## Features

- User-friendly interface
- Input fields for attendance, study hours, and previous grade
- Predict button to calculate the predicted grade
- Displays the prediction result on the webpage
