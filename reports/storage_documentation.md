# Storage Documentation

## Storage Method
Local Storage (CSV)

## Files Stored

1. student_performance_updated_1000.csv
   - Raw dataset

2. student_performance_cleaned.csv
   - Cleaned dataset after preprocessing

3. student_performance_encoded.csv
   - Encoded dataset ready for machine learning

## Reason for Choosing CSV

- Easy to use
- Lightweight
- Compatible with Python and Pandas
- Can be directly used for AI model training

## Security

The datasets are stored inside the project repository and can be retrieved whenever required.
