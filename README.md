# AI-Based Student Performance Prediction

## 📌 Project Overview

This project predicts student academic performance using Artificial Intelligence and Machine Learning techniques. It demonstrates the complete workflow of an AI application, including data collection, preprocessing, backend development, frontend interface, and documentation.

---

## 🎯 Objectives

- Collect and preprocess student performance data.
- Clean and validate the dataset.
- Store processed datasets efficiently.
- Build a backend server using Flask.
- Design a simple frontend interface.
- Prepare documentation for deployment and API communication.

---

## 📁 Folder Description

- script.js

Contains project documentation such as architecture diagrams, API documentation, and data cleaning reports.
